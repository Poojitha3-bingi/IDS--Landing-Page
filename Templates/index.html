NSL-KDD Dataset

The NSL-KDD dataset is an improved version of the KDD Cup 1999 dataset, designed to address inherent problems and provide a more realistic evaluation of intrusion detection systems.

Attack Type Training Set Test Set Description
Normal 67,343 9,711 Legitimate network traffic
DoS 45,927 7,458 Denial of Service attacks
Probe 11,656 2,421 Surveillance and probing
R2L 995 2,754 Remote to local attacks
U2R 52 200 User to root attacks
